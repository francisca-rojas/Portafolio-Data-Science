#  Descubrimiento de Patrones con Aprendizaje No Supervisado

##  Descripción

Este proyecto explora el uso de técnicas de aprendizaje no supervisado para identificar estructuras latentes y patrones en los datos.

El foco está en comprender la organización interna del dataset sin utilizar etiquetas predefinidas.

---

## Tecnologías y herramientas

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

---

##  Metodología

### 1. Preprocesamiento de datos

* Limpieza de datos
* Escalado de variables
* Transformaciones

### 2. Reducción de dimensionalidad

* PCA para visualización de estructuras
* Reducción de ruido y redundancia

### 3. Técnicas de clustering

* K-means
* Clustering jerárquico
* DBSCAN

### 4. Evaluación

* Silhouette score
* Análisis de consistencia
* Interpretabilidad estructural

---

## Resultados

Las distintas técnicas aplicadas permitieron observar diferentes estructuras en los datos, evidenciando la sensibilidad de estos métodos al preprocesamiento y a la selección de parámetros.

---

## Aprendizajes clave

* Las métricas no son suficientes para validar clustering
* La distribución de los datos influye fuertemente en los resultados
* La interpretabilidad es tan importante como el rendimiento numérico

---

## Limitaciones

* Sensibilidad a outliers
* Dependencia del preprocesamiento
* Dificultad para definir el número óptimo de clusters
